## 计划要素

在Phase的方法论中，目标是由一个个的子目标和行动组成的。而行动具有原子性，占据一定的时间并且不可被进一步分割。

- 目标

## 目标

1. 当行动横跨的时间太长时，应当认为这样的行动是一个目标。
2. 根目标是没有父目标的目标。

对没有父目标的行动，父目标被视为“生活”。
